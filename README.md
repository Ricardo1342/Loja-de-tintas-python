Loja de Tintas
Descrição

Projeto desenvolvido em Python para calcular a quantidade de tinta necessária para pintar uma área, além do custo total.

Funcionalidades
Cálculo da área (altura x largura)
Cálculo da quantidade de tinta necessária
Cálculo do número de latas
Cálculo do valor total
Tecnologias utilizadas
Python

Como executar o projeto
Clone o repositório:
git clone https://github.com/seu-usuario/loja-tintas-python
Acesse a pasta:
cd loja-tintas-python
Execute o programa:
python nome_do_arquivo.py
